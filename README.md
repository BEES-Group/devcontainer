- Doppler (<https://doppler.com/>) - Secret management

- NodeJS LTS - via FNM
- PNPM - via NodeJS's Corepack
- Eslint - via NPM
- Prettier - via NPM
- Task (<https://taskfile.dev/>) - via NPM
